# Chatbot Luma Backend (FastAPI MVP)

A minimal backend built with **FastAPI** for an emotional support chatbot inspired by Wysa.

## Features
- Emotion detection (sad, angry, anxious, happy)
- Psychologist-style responses
- Ready to connect with Next.js frontend
- Clean folder structure for academic demo

```bash
uvicorn app.main:app --reload --port 8000
